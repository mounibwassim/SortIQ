# SortIQ - Smart Recycling Waste Classification System

Smart Recycling Assistant that classifies waste from images and helps improve recycling efficiency.

## Architecture

*   **Backend**: Python / FastAPI / TensorFlow
*   **Web Dashboard**: React / Vite / TailwindCSS
*   **Mobile App**: React Native (Expo)
*   **Database**: PostgreSQL / SQLite (local fallback)

## Setup Instructions

### 1. Backend API

Requirements: Python 3.10+

```bash
cd backend
python -m venv venv
# On Windows
venv\Scripts\activate
# On Mac/Linux
source venv/bin/activate

pip install -r requirements.txt

# Create a placeholder model for local development
python create_mock_model.py

# Start the API
uvicorn main:app --reload
```

The API will be available at `http://localhost:8000`. You can view the swagger documentation at `http://localhost:8000/docs`.

### 2. Web Application

Requirements: Node.js 18+

```bash
cd web
npm install
npm run dev
```

### 3. Mobile Application

Requirements: Node.js 18+, Expo CLI

```bash
cd mobile
npx expo install
npx expo start
```

## ML Model Note

The `.h5` model included generated by `create_mock_model.py` is a randomly initialized model used *only* for API and integration testing. Real inference requires exporting the trained MobileNetV2 from the `notebooks/TCV.ipynb` environment and placing it at `backend/model/sortiq_model.h5`.

### Mobile Offline Mode
For on-device inference in the React Native app, convert the `SavedModel` to `.tflite` via `TFLiteConverter` and place it in the `mobile/assets/` folder. Use the `expo-tflite` library to run it.
